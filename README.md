# Learn Lua in Neovim

Practice files for the **Learn Lua in Neovim** video series — 32 episodes from beginner to advanced.

Learn Lua while coding in Neovim. Every keystroke is shown on screen with 3-second pauses so you can follow along at your own pace.

## Episodes

| # | Topic | File | Key Concepts |
|---|-------|------|-------------|
| 1 | Hello Lua | `episode01/hello.lua` | `print()`, strings, `:!lua %` |
| 2 | Variables and Types | `episode02/variables.lua` | `local`, `type()`, string, number, boolean, nil |

## How to Use

1. Open the file for the episode you're watching:
   ```
   nvim episode01/hello.lua
   ```
2. Follow along with the video, or use the file as a reference after watching.
3. Run any Lua file from inside Neovim:
   ```
   :!lua %
   ```
